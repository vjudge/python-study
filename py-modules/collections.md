# collections


### NamedTuple

#### 导入
```python
from collections import namedtuple
```

```python
# 创建名字为 Person 的 NamedTuple 实例 Person
Person = namedtuple('Person', ['id','name','age'])
Person(3, 'xiaoming', 19)
```
